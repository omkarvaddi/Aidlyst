# Friday Local Jarvis

Friday is a premium local command cockpit for Aidlyst. It runs at port 8787 and provides fast voice-first operations plus deeper Obsidian-brain context retrieval.

## Run locally

```bash
cd Friday
npm install
npm run dev
```

Open http://127.0.0.1:8787/

## What it does

- Voice input with browser speech recognition when available.
- Spoken response summaries with browser speech synthesis.
- Fast Ops mode for sub-second operational answers.
- Deep Brain mode for strategy, compliance, launch, affiliate, Shopify, and development planning.
- Load Brain button for selecting an Obsidian vault folder of markdown files locally.
- Premium desktop Jarvis-style UI.

## Aidlyst operating rules baked in

- Aidlyst is affiliate-first.
- Product CTAs should be View deal or Visit partner.
- Avoid cart/checkout unless Aidlyst becomes seller of record after compliance review.
- Do not provide medical advice, diagnosis, treatment, prescriptions, emergency guidance, or unsupported medical/FDA claims.
- Track outbound partner clicks with aidlyst_product_link_click.
